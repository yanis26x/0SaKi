# 0SaKi26x
Application mobile développée avec **Expo Go**.

## Lancer le projet

avoir **Node.js** installé (version utilisée : `v20.19.0`).

```bash
# Installer les dépendances
npm install

npx expo install expo-av

# Démarrer le projet
npx expo start
```
